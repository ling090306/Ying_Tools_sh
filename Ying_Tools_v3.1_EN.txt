━━━━━━━━━━━━━━━━━━ Welcome ━━━━━━━━━━━━━━━━━━
Welcome to Ying Tools! This is a partition management tool tailored for Android devices,
hoping to become a helpful assistant in maintaining your device.

━━━━━━━━━━━━━━━━━━ Script Overview ━━━━━━━━━━━━━━━━━━
Ying Tools v3.1 is a full-featured partition management script for Android devices,
running in Termux terminal with root privileges. Main features include:
  • Flash partitions (with A/B slot selection)
  • Backup single partition (with A/B slot selection)
  • Full firmware backup (automatically excludes data partitions,
    pre-checks space, generates backup info file)

━━━━━━━━━━━━━━━━━━ Usage Tips ━━━━━━━━━━━━━━━━━━
• Always back up your important data and ensure sufficient battery
  before proceeding.
• Full firmware backup skips data partitions by default (userdata,
  sdcard, etc.), but you can customize the exclude list.
• For any issues, feel free to provide feedback, but please
