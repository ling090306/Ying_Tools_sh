# Ying Tools v3.1 - Android 全功能分区管理脚本

欢迎使用 **Ying Tools**！本脚本专为 Android 设备设计，运行于 **Termux** 终端，需要 **root 权限**。它提供了安全、交互式的分区操作，支持 A/B 槽位选择，并能一键备份全部分区（排除数据分区）生成备份信息文件。

---

## 📦 功能特性

| 功能 | 说明 |
|------|------|
| **刷写分区** | 输入分区名（如 `boot`），自动检测设备是否支持 A/B，支持选择仅刷写 仅`a`、仅 `b` 或 `ab` 双槽位。 |
| **备份单个分区** | 类似刷写，可选择槽位，备份为 `.img` 文件。 |
| **全字库备份** | 扫描所有可读分区（排除虚拟设备），默认排除数据分区（`userdata`、`sdcard`等），备份前计算总大小并与可用空间对比，空间不足时警告。备份完成后生成 `Ying-tool备份.txt` 记录详细信息。 |
| **USB调试开关** | 一键开启/关闭 ADB 调试。 |
| **重启到 Fastboot** | 快速重启至 bootloader 模式。 |

---

## 🚀 使用方法

1. 将脚本文件（如 `Ying-Tools_v3.1.sh`）放置到 Termux 可访问的目录（例如 `/sdcard/`）。
2. 打开 Termux，输入 `su` 获取 root 权限。
3. 赋予脚本执行权限：  
   `chmod +x /sdcard/Ying-Tools_v3.1.sh`
4. 运行脚本：  
   `sh /sdcard/Ying-Tools_v3.1.sh`
5. 阅读风险提示并输入 `y` 同意。
6. 查看设备信息后，输入 `y` 进入主菜单。
7. 根据菜单提示选择操作。

---

## 📁 备份输出示例

执行全字库备份后，将在 `/sdcard/Ying-backup/` 生成：

- `分区名.img` - 每个成功备份的分区镜像
- `分区名.img.md5` - MD5 校验文件（若系统支持）
- `Ying-tool备份.txt` - 备份时间、设备信息、成功/失败列表等
